SyncLeetcode

Pushes your code to GitHub when a submission is Accepted on leetcode or leetcode.cn.

GitHub access

Create a new token: New classic token · New fine-grained token
or Browse GitHub → Settings → Developer settings → Personal access tokens.

  • Classic: enable repo (or public_repo for public repos only).
  • Fine-grained: Contents: Read and write on the repository you use below.

Target repository

Create the repository on GitHub first.
Empty or private repo is fine.

The OWNER in github.com/OWNER/repo.

Same name as on GitHub — the repo part of github.com/owner/repo.

Must already exist (often main).

Files inside the repo

Subfolder under repo root, e.g. python → python/1-two-sum.py.

Behavior

Success/failure alerts use your operating system’s notifications .

Allow notifications for your browser (e.g. Chrome, Edge) in system settings.

Settings saved

Enjoy coding on LeetCode 🚀

You can add your repository (owner + name) anytime in settings below.
